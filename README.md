# Process Mining Group 5

## Setup

1. Create a venv: `python3 -m venv venv`
2. Activate the venv: `source venv/bin/activate`
3. Install the requirements: `python -m pip install -r requirements.txt`
4. Install Graphviz: https://graphviz.org/download/
5. Add Graphviz to the environment variables: `C:\Program Files\Graphviz\bin`
